##  Getting Started

### 1. Clone Repository

```bash
git clone https://github.com/junaidaurungzaib-ai/portflio/.git
```

### 2. Navigate to Project
cd portfolio

## Customization
You can easily customize:
* Replace images in `/images`
* Update personal information in `about.html`
* Modify colors in `style.css`
* Add new projects in `projects.html`
